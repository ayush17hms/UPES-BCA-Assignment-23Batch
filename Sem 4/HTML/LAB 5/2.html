Overflow : visible

By default, the overflow is visible, meaning that it is not clipped and it renders outside the element's box:

background-color: powderblue;
width: 250px;
height: 65px;
border: 2px solid;
Overflow: visible;

Overflow: hidden

with the hidden value, the overflow is clipped, and the content is hidden:

you can use the overflow property when you want to have better control of the layout. The overflow property specifies what happens if content.

Overflow: scroll

Setting the overflow value to scroll, the overflow is clipped and a scrollbar is added to scroll inside the box.
